# 🍦 GELATICO 젤라티코 — 창업 안내 웹 서비스

> 바이브 코딩 실습 프로젝트 | Python + Streamlit + Plotly

## 📌 프로젝트 소개

젤라티코(Gelatico) 프랜차이즈 **신규 창업자**를 위한 창업 안내 웹 서비스입니다.
브랜드 소개부터 실제 가상 매출 데이터 시각화, 운영 시뮬레이션까지 한 곳에서 확인할 수 있습니다.

---

## 🗂️ 탭 구성

| 탭 | 내용 |
|---|---|
| 🍦 젤라티코란? | 브랜드 소개, 시그니처 메뉴, 시즌 매출 패턴 |
| 📣 마케팅 | 전략별 효과 비교, 운영 시뮬레이션, 비용 구성 차트 |
| 📊 매출현황 | 상위 5개 매장, 월별 36개월 추이, 상권별 비교, 연도별 성장 |
| 📋 창업안내 | 창업 절차 8단계, 초기 비용 테이블, 상권 가이드, 문의 폼 |

## 🚀 로컬 실행 방법

```bash
# 1. 레포지토리 클론
git clone https://github.com/YOUR_USERNAME/gelatico-franchise.git
cd gelatico-franchise

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 앱 실행
streamlit run app.py
```

---

## ☁️ Streamlit Cloud 배포 방법

1. [share.streamlit.io](https://share.streamlit.io) 접속
2. GitHub 계정 연동
3. **New app** → 레포지토리 선택
4. Main file path: `app.py`
5. **Deploy** 클릭
